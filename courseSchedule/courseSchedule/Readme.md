# 操作步骤
1. 找到GUIUtil的文件  
   ![img_1.png](img_1.png)
2. 在GuiUtil中找到标注的todo并按指示填入年月日  
![img_2.png](img_2.png)
3. 找到classSchedule文件
![img_3.png](img_3.png)
4. 在classSchedule中找到标注的todo并按指示修改修改路径
![img_4.png](img_4.png)
5. 找到App  
![img_5.png](img_5.png)
6. 将已经编辑好的课程文本复制
![img_7.png](img_7.png)
7. 在App中点击最左侧的绿色三角箭头启动程序  
![img_6.png](img_6.png)
8. 看到弹出的窗口，并在窗口中将复制好的文本粘贴，并点击回车键
![img_8.png](img_8.png)
9. 看到下方控制台输出"课程表.xls 成功生成！"即可
![img_9.png](img_9.png)
10. 然后去刚才第四步中自己设置的路径中找到课程表文件即可
